# AGENTS.md

## Product
This repository is a content automation system for a Russian-speaking AI creator who promotes referral links through useful educational content.

## Non-goals
- No spam
- No unsolicited DMs
- No fake engagement
- No scraping of private user data
- No bypass of platform restrictions

## Coding principles
- Prefer simple readable Python
- Modular architecture
- Strong typing
- Isolated adapters for external services
- Environment variables for secrets
- Good README and examples
- Avoid breaking existing interfaces

## Content rules
- Output language: Russian
- Style: natural, modern, human, not robotic
- Avoid repetitive hooks
- Avoid exaggerated promises
- Preserve creator voice where examples exist

## Workflow
- Import content
- Generate drafts
- Allow human approval
- Publish only approved content
- Save analytics metadata

## Priorities
1. Reliability
2. Ease of editing drafts
3. Batch processing of old Pinterest content
4. Clean publisher adapters
5. Analytics for CTA performance
