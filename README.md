# PROJECT: PINTEREST_REFERRAL_AGENT

## Referral Content Agent (MVP)

Простой веб-проект на FastAPI для новичка.

Он помогает:
- загружать Pinterest-пины через CSV,
- импортировать изображения без CSV,
- загружать несколько JPG/PNG,
- загружать один ZIP с JPG/PNG,
- автоматически создавать записи пинов,
- смотреть импортированные пины в браузере,
- генерировать черновики для Telegram и ВКонтакте,
- вручную одобрять или отклонять черновики.

> Сейчас нет автопубликации в соцсети и нет сложной аналитики. Данные хранятся в JSON-файлах.

> VERSION: IMAGE_IMPORT_ENABLED

---

## 1) Установка (Windows, первый запуск)

1. Откройте папку проекта.
2. Запустите файл `setup_windows.bat`.
3. Дождитесь завершения установки зависимостей.

Что делает `setup_windows.bat`:
- проверяет Python,
- создает виртуальное окружение `.venv`,
- устанавливает зависимости из `requirements.txt`.

---

## 2) Запуск проекта (Windows)

1. Запустите файл `run_api.bat`.
2. Дождитесь сообщения про запуск сервера.
3. Откройте в браузере:

`http://127.0.0.1:8000`

---

## 3) Как работать через браузер

### Вариант А. Импорт CSV
1. Перейдите в меню **«Импорт пинов»**.
2. Нажмите **«Выберите CSV-файл»**.
3. Выберите CSV с колонками:

`pin_id,title,description,board,keywords,referral_link`

4. Нажмите **«Загрузить CSV»**.

### Вариант Б. Импорт изображений без CSV
На странице **«Импорт пинов»** доступен импорт изображений:

- несколько JPG/PNG
- один ZIP с JPG/PNG

Можно указать:
- общий `board`
- общий `referral_link`

После загрузки система автоматически создаёт записи пинов.

---

## 4) Генерация черновиков

1. После импорта откройте список пинов.
2. У нужного пина нажмите **«Сгенерировать черновик»**.
3. Откроется страница черновика.

Для каждого черновика доступны:
- Telegram-версия
- VK-версия
- 3 варианта хука
- 2 варианта CTA
- выбор лучшего хука
- выбор лучшего CTA
- копирование текста

---

## 5) Модерация черновиков

На странице **«Черновики»** можно:
- просматривать черновики
- одобрять
- отклонять
- выбирать лучший хук
- выбирать лучший CTA

---

## 6) Частые ошибки

### Ошибка: Python не найден
Что делать:
1. Установите Python 3.11+.
2. При установке включите опцию **Add Python to PATH**.
3. Повторите запуск.

### Ошибка: сайт не открывается
Что делать:
1. Убедитесь, что сервер действительно запущен.
2. Проверьте адрес:

`http://127.0.0.1:8000`

### Ошибка: изображения не импортируются
Что делать:
1. Проверьте, что вы загружаете JPG, PNG или ZIP с такими файлами.
2. Попробуйте сначала загрузить 2-3 изображения для теста.

---

## 7) Где хранятся данные

- Импортированные пины: `data/pins.json`
- Черновики: `data/drafts.json`
- Загруженные CSV: `data/uploads/`
- Загруженные изображения/архивы: внутри data-потока импорта проекта
