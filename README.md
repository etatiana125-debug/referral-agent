# Referral Content Agent (MVP)

Простой веб-проект на FastAPI для новичка.

Он помогает:
- загрузить CSV со старыми Pinterest-пинами,
- посмотреть импортированные пины в браузере,
- по кнопке сгенерировать черновики для Telegram и ВКонтакте,
- вручную одобрить или отклонить черновики.

> Сейчас нет автопубликации в соцсети и нет сложной аналитики. Данные хранятся в JSON-файлах.

---

## 1) Установка (Windows, первый запуск)

1. Откройте папку проекта.
2. Запустите файл `setup_windows.bat` (двойной клик).
3. Дождитесь сообщения **«Готово! Теперь запустите run_api.bat»**.

Что делает `setup_windows.bat` автоматически:
- проверяет Python,
- создает виртуальное окружение `.venv`,
- активирует его,
- устанавливает зависимости из `requirements.txt`.

---

## 2) Запуск проекта (Windows)

1. Запустите файл `run_api.bat` (двойной клик).
2. Дождитесь сообщения про запуск сервера.

> Если `.venv` не найден, скрипт подскажет сначала запустить `setup_windows.bat`.

---

## 3) Открытие в браузере

Откройте адрес:

`http://127.0.0.1:8000`

Дополнительно:
- Swagger API: `http://127.0.0.1:8000/docs`

## 4) Как работать через браузер

### Шаг 1. Импорт CSV
1. Перейдите в меню **«Импорт пинов»**.
2. Нажмите **«Выберите CSV-файл»**.
3. Выберите ваш файл с колонками:
   `pin_id,title,description,board,keywords,referral_link`
4. Нажмите **«Загрузить CSV»**.

### Шаг 2. Генерация черновиков
1. В таблице пинов нажмите **«Сгенерировать черновик»** у нужного пина.
2. Вас перенаправит на страницу конкретного черновика.

### Шаг 3. Просмотр вариантов на странице черновика
На странице черновика вы увидите отдельно:
- Telegram-черновик
- VK-черновик
- 3 варианта хука
- 2 варианта CTA

Там же есть кнопки:
- **«Скопировать Telegram-текст»**
- **«Скопировать VK-текст»**

### Шаг 4. Выбор лучшего варианта
1. В блоке выбора отметьте один лучший хук и один лучший CTA.
2. Нажмите **«Сохранить выбранные варианты»**.
3. Вернитесь в список **«Черновики»** — там будет видно, выбран ли хук и CTA.

### Шаг 5. Модерация черновиков
На странице черновика используйте кнопки:
- **«Одобрить»**
- **«Отклонить»**

## 5) Частые ошибки

### Ошибка: Python не найден
Что делать:
1. Установите Python 3.11+ с официального сайта.
2. При установке включите опцию **Add Python to PATH**.
3. Перезапустите `setup_windows.bat`.

### Ошибка: зависимости не установились
Что делать:
1. Проверьте интернет.
2. Если вы в корпоративной сети — проверьте прокси/фаервол.
3. Запустите `setup_windows.bat` повторно.

### Ошибка: run_api.bat пишет, что нет .venv
Что делать:
1. Сначала запустите `setup_windows.bat`.
2. Затем снова `run_api.bat`.

## 6) Где менять стиль текстов вручную

Если хотите подправить тон и формулировки:
- `prompts/hooks_library.txt` — библиотека начал (хуков) для вариативности первых абзацев.
- `prompts/cta_library.txt` — библиотека мягких CTA для концовки поста.
- `prompts/shared_rules.txt` — общие правила качества текста и структура.
- `prompts/telegram_style.txt` — стиль Telegram (живой, тёплый, разговорный).
- `prompts/vk_style.txt` — стиль VK (более структурный, удобный для ленты).
- `prompts/author_voice.txt` — общий авторский тон.

Все эти файлы можно редактировать вручную в любом текстовом редакторе — Python-код трогать не нужно.
